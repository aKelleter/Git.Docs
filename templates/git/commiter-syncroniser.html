1. Commiter les modifications


git add .
git commit -m "Message de commit"
# Si vous ommetez le paramètres -m GIT ouvrira l'éditeur qui est enregistré dans la configuration

2. Publier les modifications vers le dépôt distant


git push
# Si vous ommettez le nom de la branche, GIT publiera vers la branche
# actuellement active. Si vous souhaitez publier vers une autre branche,
# vous pouvez spécifier le nom de la branche comme ceci :
git push origin nom_de_la_branche

3. Connecter un dépôt local à un dépôt distant


git remote add origin 
git branch -M main
git push -u origin main
# Si vous avez déjà un dépôt local et que vous souhaitez le connecter à un dépôt distant,
# vous pouvez utiliser la commande suivante :
# git remote add origin 
# Ensuite, vous pouvez pousser vos modifications vers le dépôt distant avec :
# git push -u origin main

4. Mettre à jour le dépôt local avec le dépôt distant


git pull origin main
