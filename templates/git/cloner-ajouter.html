1. Créer un répertoire de travail


mkdir /path/to/your/project

# Se placer dans le répertoire
cd /path/to/your/project

2. Initialiser un dépôt Git


# Initialiser un dépôt Git
git init

3. Ajouter un dépôt distant Github


# Ajouter un dépôt distant
git remote add origin https://github.com/username/repo.git

4. Afficher les dépôts distants


# Afficher les dépôts distants
git remote -v

5. Ajouter un fichier


# Ajouter un fichier
git add <nom_du_fichier>
# Exemple : git add index.html

6. Valider les modifications


# Valider les modifications
git commit -m "Message de validation"

7. Pousser les modifications


# Pousser les modifications
git push -u origin main

8. Cloner un dépôt distant


# Cloner un dépôt distant
git clone https://github.com/username/repo.git

9. Mettre à jour l'URL d'un dépôt distant


# Mettre à jour l'URL d'un dépôt distant
git remote set-url origin https://github.com/username/new-repo.git
